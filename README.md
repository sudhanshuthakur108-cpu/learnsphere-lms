# 🎓 LearnSphere – Learning Management System

LearnSphere is a modern and responsive Learning Management System (LMS) built using React.js and Tailwind CSS.  
The platform provides separate dashboards for students and educators, allowing course management, enrollments, and interactive learning experiences through a clean and user-friendly interface.

🔗 Live Demo: https://learnspheere.netlify.app/

---

# ✨ Features

## 👨‍🏫 Educator Dashboard
- Add and manage courses
- View enrolled students
- Track total enrollments
- Dashboard analytics and earnings section
- Course listing management

## 👨‍🎓 Student Dashboard
- Browse available courses
- Enroll in courses
- View enrolled courses
- Continue learning interface
- Responsive course cards and layouts

## 🔐 Authentication
- User authentication using Clerk
- Profile integration
- Protected dashboard routes

## 🎨 UI & UX
- Fully responsive modern design
- Clean dashboard layouts
- Tailwind CSS styling
- Reusable React components
- Interactive navigation

# 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/sudhanshuthakur108-cpu/learnsphere-lms.git
```

Go to project directory:

```bash
cd learnsphere-lms/client
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
